# Icons Directory

This directory should contain the tab bar icons:

- `home.png` - Home tab icon (16x16px recommended)
- `home-active.png` - Home tab active icon
- `feed.png` - Feed tab icon
- `feed-active.png` - Feed tab active icon
- `profile.png` - Profile tab icon
- `profile-active.png` - Profile tab active icon

You can use any icon library or create your own icons. Recommended size: 16x16px or 24x24px PNG format with transparent background.

## Icon Sources
- [Feather Icons](https://feathericons.com/)
- [Iconfont](https://www.iconfont.cn/)
- [Material Icons](https://fonts.google.com/icons)
- Or create custom icons using Figma/Sketch
